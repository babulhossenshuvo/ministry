# Bangladesh UN Portal Prototype

An official, premium web portal representing the **People's Republic of Bangladesh at the United Nations**. This prototype demonstrates a production-quality, responsive, and multilingual interface designed for diplomatic and official use.

## 🚀 Key Features

- **Foreign Secretary Portfolio**: A complete profiling section for the Foreign Secretary, featuring biography, achievements, and career timeline.
- **Multilingual Support**: Supports 7 major languages (English, Bengali, Arabic, Chinese, Spanish, French, and Portuguese) with real-time switching and RTL support for Arabic.
- **Mission & Engagement**: Showcases Bangladesh's active role at the UN, including peacekeeping contributions and resolutions.
- **Interactive Delegation**: A searchable and filterable directory of official mission members.
- **Interactive Maps**: Diplomatic mission and ministry locations integrated with interactive Leaflet maps.
- **Dark & Light Mode**: A sophisticated theme system with seamless transitions.
- **Premium Design**: Built with a strict design system (Lush Green & Gold) using Tailwind CSS and Framer Motion for high-quality animations.

## 🛠️ Technology Stack

- **Framework**: React.js with Vite
- **Styling**: Tailwind CSS & Vanilla CSS Variables
- **Animations**: Framer Motion
- **Internationalization**: i18next & react-i18next
- **Icons**: Lucide React
- **Maps**: React-Leaflet
- **Charts**: Recharts
- **Deployment**: Static Export Ready

## 📂 Project Structure

```bash
src/
├── components/ # Atomic UI & Layout components
├── context/ # State management for Theme & Language
├── data/ # Centralized placeholder data
├── locales/ # JSON translation files (7 languages)
├── pages/ # Core page implementations
└── styles/ # Global styles and CSS variables
```

## 📜 How to Run Locally

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Start Development Server**:
   ```bash
   npm run dev
   ```
3. **Build for Production**:
   ```bash
   npm run build
   ```
